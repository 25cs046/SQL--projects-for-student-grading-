## Project Overview
A database system to manage students, subjects, and their grades with performance analysis.

## Tables
- **students** — stores student details
- **subjects** — stores subject information
- **grades** — stores marks per student per subject

## Features
1. All students with their grades (JOIN)
2. Average marks per student (ROUND + AVG)
3. Top performers using HAVING clause
4. Subject-wise average, highest and lowest marks
5. Failed students (marks below 40)

## Concepts Used
- JOIN across 3 tables
- AVG, MAX, MIN, ROUND
- GROUP BY, HAVING
- WHERE with conditions
- ORDER BY ASC/DESC

## Tool Used
SQLite
